## to run the example

requirements:
- node.js (install from [http://nodejs.org](http://nodejs.org))
- git

1. Clone this repo
```console
$ git clone https://github.com/jden/open-source-map-making.git
$ cd open-source-map-making
```
2. Install dependencies
```console
$ cd leaflet-example
$ npm install
```

3. Start the demo
```console
$ npm start
```
